Hermes Trading Framework

A modular, autonomous trading framework where strategies are black boxes and Hermes learns portfolio-level allocation.

Python 3.11+ Alpaca Interactive Brokers YAML Config Deterministic Risk

Architecture

Market Data (live or CSV) → Regime Detection → Hermes (Allocation) → Risk Veto → Broker (Alpaca / Mock)
 Strategies (black-box) → Signals → Orders

Key Principles

  • ✓ Single execution path — paper vs live is a broker endpoint choice, not a runtime mode
  • ✓ Strategies are black boxes — they generate signals, not allocate capital
  • ✓ Hermes allocates — it decides when and how much to deploy each strategy
  • ✓ Risk is absolute — deterministic veto layer that Hermes cannot override
  • ✓ All decisions are logged — full audit trail for every allocation, veto, and fill
  • ✓ No hybrid brokers — engine rejects CSV market data combined with a live broker

Modules

⚙ Core

Shared types, event bus, and clock. The foundation every module depends on.

★ Strategies

Black-box strategy implementations: SMA crossover, RSI mean reversion, pullback continuation, VWAP reversion, and more.

⚖ Hermes

Portfolio allocation agent with scoring, conflict resolution, position sizing, and multi-agent coordination.

⚠ Risk

Deterministic risk veto layer with drawdown ladders, FTMO guards, and profile transitions.

▶ Execution

Broker interface with Alpaca, Interactive Brokers, and mock implementations. Safety guards included.

⚙ Market

Data loading, regime detection, live feeds, CSV adapters, and streaming buffers.

📊 Analytics

Performance analytics engine with trade metrics and portfolio analysis.

🚀 Meta

Policy evolution, drift detection, LLM tuning, and strategy mutation for adaptive optimization.

📈 Visualization

Decision annotations, correlation context, price charts, and audit logging for observability.

Quick Start

# Install
pip install -e .

# Local replay with MockBroker
python scripts/run_demo.py

# Production (Alpaca — paper or live via env vars)
python scripts/run_engine.py

# Observability dashboard
pip install streamlit pandas plotly pyyaml
streamlit run dashboard/app.py
